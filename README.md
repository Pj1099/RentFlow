# Rental Management System

A comprehensive MERN stack application for managing rental businesses with quotations, orders, invoicing, and inventory management.

## Features

- **Multi-role System**: Customer, Vendor, and Admin access
- **Product Management**: Rentable products with flexible pricing
- **Quotation & Order Flow**: Complete rental lifecycle
- **Inventory Management**: Real-time availability tracking
- **Invoicing**: Automated invoice generation with payment support
- **Pickup & Return**: Track rental lifecycle
- **Dashboards & Reports**: Business insights and analytics

## Tech Stack

- **Frontend**: React.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT

## Installation

1. Install dependencies:
```bash
npm run install-all
```

2. Configure environment variables in `.env` file

3. Start MongoDB server

4. Run the application:
```bash
npm run dev
```

## Default Ports

- Backend: http://localhost:5000
- Frontend: http://localhost:3000

## User Roles

- **Customer**: Browse products, create quotations, place orders
- **Vendor**: Manage products, process orders, create invoices
- **Admin**: Full system access, manage users and settings
